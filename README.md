# 康正官网视觉 Demo V2

这是独立于旧版 Demo 的 React / Vite 视觉原型，重点验证：

- 沉浸式全屏首屏与浮动胶囊导航
- 圆角叠层转场与暖色品牌叙事
- 桌面端粘性导航 + 滚动生活项目卡片
- 分角色建联窗口与移动端体验
- `prefers-reduced-motion` 弱动效降级

## 本地启动

```bash
npm install
npm run dev
```

## 素材说明

- `public/media/kangzheng/`：15 张经过网页压缩的本地 JPEG，全部由 GPT Image 2 生成，不依赖境外图床。
- `../source-assets/gpt-image2/`：同一批素材的原始 PNG 源文件。
- 四张核心阶梯卡严格对应：`10a 健康自主`、`10b 数字自主`、`10c 价值创造`、`10d 居家独立`。
- 其余素材覆盖出行、社交、教练、家庭、机构、城市、成果记录、项目场地与当代书法抽象。
- `15-brand-symbol-concept` 是视觉概念，不是可直接注册使用的正式商标。定稿前仍须完成商标检索、原创性复核与矢量重绘。

生成素材仅用于视觉原型。正式官网上线应优先替换为经过授权的真实人物、教练、服务现场与生活成果影像，并保留本地静态降级图。

本 Demo 不会发送或保存联系表单内容，也不构成服务、医疗、投资或收益承诺。
